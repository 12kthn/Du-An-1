Hướng dẫn cài đặt phần mềm

Yêu cầu cài đặt sẵn SQL Sever phiên bản 2008 trở lên và jdk 1.5 trở lên

B1: Giải nén file PS08445_Polypro.rar
B2: Cài đặt CSDL từ file csdl.sql trong thư mục db
B3: Chạy file install.exe trong thư mục gốc để cài đặt chương trình
Phần mềm được cài đặt trong C:\Program Files (x86)\PS08445_Polypro
Mở file PS08445_Polypro.exe để khởi động phần mềm
